GMT中文手冊¶

歡迎來到 Generic Mapping Tools (GMT) 的世界。GMT 是地球科學最廣泛使用的製圖軟體之一。

本文件是由 GMT中文社區 維護整理的GMT中文手冊。手冊詳盡地介紹了GMT的用法並提供了大量的實用示例。其既可以作爲初學者的入門讀物，也可以作爲日常使用的參考書。

文件源碼在 GitHub 上開源，歡迎更多的志願者加入到文件的維護工作中來。
