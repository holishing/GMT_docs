標準選項¶

GMT 有100多個模塊，每個模塊的具體效果由模塊的衆多選項來決定。不同的模塊有不同的選項，這其中有一些選項是標準選項，即這些選項在所有的命令裏都具有完全相同的意義。故而把這些通用的選項單獨拿出來介紹。

GMT 中的選項都是以 - 加一個字符的形式構成，通常這個字符是經過精心挑選的，使得用戶很容易根據字符記住該選項的作用。

本章將介紹 GMT 中標準選項的用法。由於標準選項在所有 GMT 模塊中的用法是一樣的，所以在模塊手冊中介紹每個模塊的具體語法時，將不再詳細介紹這些標準選項。

GMT標準選項列表¶
選項 說明
-B 定義底圖邊框和軸的刻度、標註、標籤等屬性
-J 選擇地圖投影方式或座標變換
-R 指定區域範圍
-U 在圖上繪製時間戳
-V 詳細報告模式
-X 移動X方向上的繪圖原點
-Y 移動Y方向上的繪圖原點
-a 將非空間數據與某些列聯繫在一起
-b 控制二進制的輸入或輸出
-c 選擇特定的子圖
-d 將輸入或輸出中的 nodata 替換成NaN
-e 僅處理與指定 pattern 匹配的數據
-f 設置ASCII輸入或輸出的格式
-g 根據數據間斷對數據進行分段
-h 跳過數據的文件頭段記錄
-i 選擇輸入列
-j 確定球面距離的計算方式
-l 爲當前繪製的符號或線條添加圖例記錄
-n 設置網格插值方式
-o 選擇輸出列
-p 控制3D視角圖
-r 設置網格配準方式
-s 控制NaN記錄的處理方式
-t 設置圖層透明度
-x 設置並行的核數（僅限於支持並行的模塊）
-: 輸入數據是 y/x 而不是 x/y
